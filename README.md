Project overview:
The repository contains the CSE 476 final project. The agent uses bens_model via the provided API and 3 inference time techniques ( CoT, self consistency and reflection).

Enviroment Setup:
everything was run on final_project_tutorial.ipynb on VS Code Jupyter environment using any Python kernel

How to run:
Open folder in VS code
start Jupyter server and use Python kernel
open final_project_tututial.ipynb
Run all notebook cells from top to bottom:
All the code till agent, once you run agent run the next cell block and wait for it to finish.
Once done run the block about merging the files
